# 部署說明 — Charlene's Book List

整個流程分三大步驟，完成後就有一個可以從任何電腦存取的網路書單。

---

## 步驟一：部署 Cloudflare Worker（後端 API + 資料庫）

### 1-1. 安裝 Wrangler 工具

打開 Terminal，進入 worker 資料夾：

```bash
cd ~/Dropbox/俞晰/Reading\ Book\ List/charlene-books/worker
npm install
```

### 1-2. 登入 Cloudflare

```bash
npx wrangler login
```

這會開啟瀏覽器，用您的 Cloudflare 帳號登入後關閉即可。

### 1-3. 建立 D1 資料庫

```bash
npx wrangler d1 create charlene-books
```

執行完後會看到類似這樣的輸出：

```
✅ Successfully created DB 'charlene-books'

[[d1_databases]]
binding = "DB"
database_name = "charlene-books"
database_id = "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"   ← 複製這個 ID
```

**把那個 database_id 複製起來，然後更新 wrangler.toml 檔案**，把 `REPLACE_WITH_YOUR_DATABASE_ID` 換成真實的 ID。

### 1-4. 初始化資料庫結構

```bash
npx wrangler d1 execute charlene-books --remote --file=schema.sql
```

### 1-5. 部署 Worker

```bash
npx wrangler deploy
```

部署成功後會顯示：

```
✅ Deployed charlene-books-api
   https://charlene-books-api.你的名字.workers.dev
```

**把這個網址記下來**，等一下網頁設定會用到。

---

## 步驟二：把網頁放到 GitHub

### 2-1. 在 GitHub 建立新 repo

1. 前往 https://github.com/new
2. Repository name 填 `charlene-books`
3. 選 **Public**（Cloudflare Pages 免費版需要公開 repo，但書單本身沒有敏感資料）
4. 點 **Create repository**

### 2-2. 把檔案推上去

在 Terminal 執行（把 `YOUR_GITHUB_USERNAME` 換成您的 GitHub 帳號名稱）：

```bash
cd ~/Dropbox/俞晰/Reading\ Book\ List/charlene-books

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/charlene-books.git
git push -u origin main
```

---

## 步驟三：在 Cloudflare Pages 部署網頁

### 3-1. 建立 Pages 專案

1. 前往 https://dash.cloudflare.com
2. 左側選單點 **Workers & Pages**
3. 點 **Create** → **Pages** → **Connect to Git**
4. 選擇您的 `charlene-books` repo
5. 設定如下：
   - **Build command**：（留空）
   - **Build output directory**：`web`
6. 點 **Save and Deploy**

部署完成後會得到類似這樣的網址：
```
https://charlene-books.pages.dev
```

---

## 步驟四：設定網頁連接 Worker

1. 用瀏覽器開啟您的 Pages 網址
2. 點右上角 **⚙️ 設定**
3. 輸入步驟 1-5 的 Worker 網址，例如：
   ```
   https://charlene-books-api.你的名字.workers.dev
   ```
4. 點「儲存並連線」

完成！書單就可以從任何裝置存取了。

---

## 步驟五（選擇性）：匯入現有書單

網頁設定好之後：
1. 點右上角 **📥 匯入 xlsx**
2. 選擇 `Charlene's Book List.xlsx`
3. 確認匯入（會覆蓋雲端現有資料）

---

## 之後更新網頁

如果網頁需要修改，只要：

```bash
cd ~/Dropbox/俞晰/Reading\ Book\ List/charlene-books
git add .
git commit -m "更新說明"
git push
```

Cloudflare Pages 會自動重新部署。

---

## 常見問題

**Q: 從不同電腦要怎麼用？**
A: 直接在瀏覽器開啟您的 Pages 網址，點 ⚙️ 設定輸入 Worker 網址一次，之後就會記住。

**Q: Worker 網址會一直是同一個嗎？**
A: 是的，部署後不會變。

**Q: 資料會遺失嗎？**
A: 資料存在 Cloudflare D1，非常穩定。建議定期用「📤 匯出 xlsx」功能備份一份在本地。
